= Aktualisieren Sie die AI Data Engine software
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Als Speicheradministrator können Sie die AI Data Engine (AIDE)-Software, die NetApp DCN-Firmware und andere Systemdateien auf Ihrem AIDE-System mit ONTAP System Manager aktualisieren.

AIDE unterstützt die automatische Software-Update-Funktionalität von ONTAP nicht. Sie können sich für Benachrichtigungen von link:https://mysupport.netapp.com/site/downloads["NetApp Support Site Downloads"^] anmelden, aber alle Updates der AIDE software werden manuell vom Administrator durchgeführt.

.Über diese Aufgabe
Das kombinierte AIDE-Softwarepaket ist deutlich größer als ein typisches ONTAP Updatepaket (AIDE-Paket ist ungefähr 40GB). Planen Sie längere Upload- und Installationszeiten ein, wenn Sie die AIDE-Software aktualisieren.

.Bevor Sie beginnen
* Sie benötigen Administratorrechte für den Speicherbereich, um NetApp DCN Firmware und AIDE Software zu aktualisieren.
* Sie benötigen Ihre NetApp Support Site-Zugangsdaten für Ihr aktives Konto.
* link:upgrades-aide-summary.html#compatibility-matrix["Stellen Sie die Kompatibilität zwischen ONTAP, NetApp DCN-Firmware und AIDE-Software sicher, wenn Sie Updates planen"].



IMPORTANT: Die Rückgängigmachungsfunktion wird für NetApp DCN-Firmware- oder AIDE-Software-Updates nicht unterstützt. Nach der Installation eines Updates ist ein Downgrade auf eine vorherige Version nicht mehr möglich.

.Schritte
. https://mysupport.netapp.com/site/products/all/details/aide/downloads-tab["Laden Sie die kombinierte NetApp DCN-Firmware- und AIDE-Software-Update-Datei auf einen lokalen Client herunter"^].
. Im System Manager wählen Sie *Cluster > Settings > Software updates*.
. Neben *Software updates* wählen Sie image:icon_arrow.gif["blauer Pfeil, der nach rechts zeigt"].
. Unter AI Data Engine-Updates wählen Sie *Add AI Data Engine software files* aus und wählen Sie das Update-Paket aus.
. Nach Abschluss des Paket-Uploads wählen Sie *Aktualisieren*, um mit der Installation der Updates auf den NetApp DCNs zu beginnen.
+

NOTE: AIDE Console ist während des NetApp DCN Updates nicht verfügbar oder zugänglich.



.Ergebnis
NetApp DCNs werden mit AIDE-Software aktualisiert und die aktualisierte Version wird für jeden Knoten angezeigt.

.Verwandte Informationen
