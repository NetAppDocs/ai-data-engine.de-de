= Installations- und Einrichtungsworkflow für von NetApp bereitgestellte Datenrechenknoten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Zur Installation und Konfiguration Ihrer von NetApp bereitgestellten Datenrechenknoten überprüfen Sie die Hardwareanforderungen, bereiten Ihren Standort vor, installieren und verkabeln die Hardwarekomponenten, schalten das System ein und richten Ihren ONTAP Cluster ein.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]link:install-setup-requirements.html["Überprüfen Sie die Anforderungen für die Hardwareinstallation"]
[role="quick-margin-para"]
Stellen Sie sicher, dass Sie über ein vorhandenes AFX 1K-Speichersystem verfügen, und überprüfen Sie anschließend die Hardwareanforderungen für die Installation von NetApp-bereitgestellten Datenrechenknoten in Ihrer Umgebung. Informationen zur Installation des AFX 1K-Speichersystems finden Sie unter link:https://docs.netapp.com/us-en/ontap-afx/install-setup/install-setup-workflow.html["AFX 1K Speichersystem-Installationsdokumentation"^].

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]link:prepare-hardware.html["Bereiten Sie die Installation Ihrer Data Compute Nodes vor"]
[role="quick-margin-para"]
Zur Vorbereitung der Installation Ihrer NetApp Datenrechenknoten müssen Sie den Standort vorbereiten, die Umgebungs- und Stromanforderungen prüfen und sicherstellen, dass ausreichend Platz im Serverschrank vorhanden ist. Dann entpacken Sie die Geräte, vergleichen deren Inhalt mit dem Lieferschein und registrieren die Hardware, um Supportleistungen in Anspruch nehmen zu können.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drei"]link:deploy-hardware.html["Installieren Sie die Hardware für Ihre Rechenknoten"]
[role="quick-margin-para"]
Installieren Sie die Schienensätze für Ihre Datenrechenknoten. Befestigen Sie Ihre von NetApp bereitgestellten Datenrechenknoten im Schrank. Bringen Sie abschließend Kabelmanagement-Geräte an der Rückseite des Systems an, um die Kabel ordentlich zu verlegen.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Vier"]link:cable-hardware.html["Verkabeln Sie Ihre Rechenknoten"]
[role="quick-margin-para"]
Um die Hardware zu verkabeln, verbinden Sie zuerst die von NetApp bereitgestellten Datenrechenknoten mit Ihrem Datennetzwerk und anschließend die NetApp Datenrechenknoten mit den Cluster-Switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Fünf"]link:power-on-hardware.html["Schalten Sie Ihre Rechenknoten ein."]
[role="quick-margin-para"]
Nachdem Sie das Rack installiert und Ihre Hardware verkabelt haben, sollten Sie Ihre Rechenknoten und Ihre Controller-Knoten für das AFX-Speichersystem einschalten, falls diese noch nicht eingeschaltet sind.
